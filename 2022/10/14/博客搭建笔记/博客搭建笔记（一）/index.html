# 准备工作

  1. Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  2. Git

# 搭建步骤

# 一、安装 Hexo 生成网站文件

在文件夹中右击 git brash Here

$ npm install -g hexo-cli
当然如果对 npm 比较熟悉可以使用 $ npm install hexo 安装 hexo。
安装以后，可以使用以下两种方式执行 Hexo：

  1. npx hexo <command>
  2. 将 Hexo 所在的目录下的 node_modules 添加到环境变量之中即可直接使用 hexo <command>

这里我使用的是非全局安装方式搭建，安装完 hexo 后使用命令 npx hexo init <自己取名> 这里我以 belog 为例
